Definition of a Computer

Core Definition

A computer is an electronic device that operates under the control of programs stored in its own memory unit. It accepts data as input, processes it under a set of instructions (Programs), and produces the desired output (Information).

Program

A computer program is a set of related instructions written in the language of the computer, used to make the computer perform a specific task or direct it on what to do.

Data

Data is a collection of raw facts, figures, or instructions that do not have much meaning on their own. Data may be in the form of numbers, alphabets/letters, or symbols.

Types of Data

  • Digital (discrete) data — discrete in nature; represented as numbers, alphabets, or symbols. Obtained by counting (e.g., 1, 2, 3…).
  • Analogue (continuous) data — continuous in nature; represented in physical form (e.g., pressure, temperature, humidity). Obtained by measurement; output appears as smooth graphs.

Data Processing

Data processing is the process of collecting all items of data and converting them into information. Processing may involve calculation, comparison, or any other logical operation.

Information

Information is data that has been refined, summarized, and manipulated into a more meaningful form for decision-making. Information must be accurate, timely, complete, and relevant.

Characteristics of a Computer

1. Speed

Computers operate at very high speeds and can perform many functions within a very short time. Speed is measured in fractions of a second:

  • Millisecond — a thousandth of a second (10⁻³)
  • Microsecond — a millionth of a second (10⁻⁶)
  • Nanosecond — a thousand-millionth of a second (10⁻⁹)
  • Picosecond — a million-millionth of a second (10⁻¹²)

2. Accuracy

Computers are very accurate and rarely make mistakes. When errors occur, built-in self-checking features can detect and correct them. Most errors arise from incorrect data entry — hence the principle GIGO (Garbage In, Garbage Out).

3. Reliability

A computer produces the correct answer when given correct instructions and correct data. The output is only as reliable as the instructions and data supplied.

4. Consistency

Given the same data and instructions, a computer will produce the same answer every time that particular process is repeated.

5. Storage

A computer can store large amounts of data in a very small space, retrieve it when needed, and protect it from unauthorized access through passwords.

6. Diligence

Unlike humans, a computer never gets tired or bored. It performs the millionth calculation with the same speed and accuracy as the first.

7. Automation

Once given instructions, a computer carries out its job automatically until it is complete. It can also perform a variety of jobs as long as there is a well-defined procedure.

8. Versatility

A computer can be used in different places to perform a large number of different jobs depending on the instructions fed to it.

Parts of a Computer

A computer consists of a collection of interconnected components working as a single entity:

  • The System Unit
  • Input Devices
  • Output Devices
  • Storage Devices

System Unit

The system unit is the casing that houses the computer's electronic components, including the CPU and storage devices.

Components inside the System Unit

  • Central Processing Unit (CPU) — the "brain" of the computer
  • Motherboard
  • Power Supply Unit
  • Memory storage devices
  • Disk drives

Types of System Units

  • Tower style — stands alone; designed to sit on the floor; more expansion space.
  • Desktop style — lies flat on a desk with the monitor resting on top.

Features of the System Unit

  • Houses the CPU
  • Connects to all peripheral devices using ports
  • Contains the computer's power switch

Input Devices

Input devices are used to enter data into the computer. They accept data and convert it into a suitable form the computer can understand.

KeyboardMouseJoystickLight PenScanner

The Keyboard

The keyboard looks like a typewriter and has letters, numbers, and other keys through which data is entered. To enter data, the user presses the required keys.

The Mouse

A pointing device that enables the user to issue instructions to the computer by controlling a special mouse pointer displayed on the screen.

Output Devices

Output devices give the end results of data entered into the computer. They convert processed data (information) from the computer into human-sensible form.

MonitorPrinterPlotterSpeakers

The Monitor

A television-like screen used for displaying output. When you type a letter or number on the keyboard, it shows up on the monitor, enabling the user to monitor what is going on in the computer.

Printer

Printers are used to create permanent copies of output on paper.

Memory Storage Devices

Storage devices hold programs and data until they are needed for processing, and store results after processing.

Primary (Main) Storage

Storage found within the computer itself, used to hold data and programs currently being used by the processor.

RAMROM

Secondary (Backing) Storage

Used to store backup information not needed immediately. Provides permanent data storage on magnetic or optical media.

Hard DiskFloppy DiskMagnetic TapeCD/DVDZip Disk

Computer Peripherals

Peripheral devices are components connected to the system unit to assist the computer in serving its users. They connect via data interface cables and ports.

MonitorKeyboardMousePrinterModemSpeakersPlotter

History of Computing

  • Before 1900: data processing was done manually using stones and sticks.
  • ~2000 years ago: Asian merchants developed the Abacus — a rectangular frame fitted with wires and beads for calculating large figures.
  • Charles Babbage developed the Analytical Engine — regarded as the forerunner of modern computers.
  • 1939: Professor Howard Aiken of Harvard University designed Mark 1, the first computer-like machine.
Abacus — How it Works
Each bead in the lower row has a value of 1; each bead in the upper row has a value of 5. A bead moved to the crossbar represents its value; beads away from the crossbar represent zero.

Computer Generations

A computer generation is a grouped summary of gradual developments in computer technology, classified in durations rather than individual years.

1st Generation

1946 – 1956

  • Used Vacuum Tubes
  • Speed: Milliseconds
  • Max memory: ~2 KB
  • Very large, costly, unreliable
  • Examples: ENIAC, EDVAC, UNIVAC
2nd Generation

1957 – 1963

  • Used Transistors
  • Speed: Microseconds
  • RAM: up to 32 KB
  • Smaller, less costly
  • Examples: IBM 1401, UNIVAC 1107
3rd Generation

1964 – 1979

  • Used Integrated Circuits (ICs)
  • Speed: 5 MIPS
  • RAM: up to 2 MB
  • Multi-user support
  • Examples: IBM 360, IBM 370
4th Generation

1979 – 1989

  • Used LSI / VLSI circuits
  • Speed: 50 MIPS
  • RAM: hundreds of MB
  • Examples: IBM 308, Apple II
5th Generation

1990 – Present

  • Parallel architectures, 3D circuits
  • Speed: Nanoseconds/Picoseconds
  • RAM: 1 GB – 1 TB
  • Artificial Intelligence
  • Supercomputers, Microcomputers

Factors that Determine the Type of Computer

  1. Type of processor (CPU) — microcomputers use a single microprocessor; larger computers use multiple high-speed components.
  2. Processing speed — larger computers have faster clocks and process more instructions per second.
  3. Amount of Main memory (RAM) — larger computers have more RAM, supporting larger data volumes and more sophisticated programs.
  4. Storage capacity of the Hard disk — larger computers have higher storage capacities.
  5. Cost — cost is directly related to size. Microcomputers are least expensive; supercomputers are most expensive.
  6. Speed of output devices — microcomputer output speed measured in characters per second (cps); larger computers measured in lines or pages per minute (lpm/ppm).
  7. Number of users — microcomputers support 1–3 users; larger computers can support hundreds simultaneously.

Classification by Physical Size

Supercomputers

The fastest, largest, most expensive, and most powerful computers. Use multiple processors; require special cooling systems. Operated by computer specialists; can support 500+ users simultaneously.

Uses: Weather forecasting, petroleum research, defence/weapons analysis, aerodynamic simulation.

Examples: CRAY T3D, NEC-500.

Mainframe Computers

Less powerful than supercomputers but still have very high processing speeds and large storage. Support 5–300 terminals and up to hundreds of users at once.

Uses: Banks, hospitals, government departments, airline reservations, internet servers.

Examples: IBM 4381, ICL 39 Series.

Minicomputers

Smaller than mainframes; can support several users (e.g., 6 at a time). Cheaper than mainframes, more costly than microcomputers.

Uses: Scientific labs, engineering plants, insurance companies, network servers.

Example: PDP-8 (1965, Digital Equipment Corporation).

Microcomputers

Personal Computers (PCs) designed for one person at a time. Processing done by a single-chip microprocessor. Smaller, cheaper, energy-efficient, and highly reliable.

Uses: Schools, small businesses, communication centres.

Examples: Apple Macintosh, Dell, Compaq.

Laptops & Notebooks

Small and portable PCs that fit in a briefcase. Run on electricity or rechargeable batteries. Have built-in disk drives and flat LCD screens.

Palmtops

Small enough to fit in a pocket; used as personal organizers with minimal programs. Example: PDA (Personal Digital Assistant).

Desktop Computers

Designed to be used on a desk; not portable. Includes home computers, personal computers (PCs), workstations, and embedded computers.

Classification by Purpose

General-Purpose Computers

Designed to perform a wide variety of tasks using specifically written programs. Programs are exchangeable — you load the appropriate program for each task. Examples: Mainframes, minicomputers, microcomputers, laptops.

Special-Purpose Computers

Designed to handle only one specific task. Programs are fixed (hard-wired) at manufacture. Very efficient and fast for their dedicated task. Examples: Industrial robots, mobile phones, calculators, digital watches, petrol pumps, washing machine controllers, automatic pilots, word processors.

Dedicated Computers

A general-purpose computer committed to a single processing task (e.g., dedicated to word processing only).

Classification by Functionality

Digital Computers

Operate on discrete data; process both numeric and alphabetic data. Based on two states — ON/OFF or 1/0. Most widely used type. Output is in numbers, alphabets, and symbols.

Examples: Digital watches, calculators, microcomputers.

Analogue Computers

Operate on continuous data by measuring changes in physical quantities (voltage, speed, pressure, temperature, etc.). Usually special-purpose devices dedicated to a single task. Output is in the form of smooth graphs.

Examples: Flight simulators, bathroom scales, thermometers, speedometers, petrol pumps.

Hybrid Computers

Process both analogue and digital data by combining the elements of both types. More expensive. Example: Hospital ICU systems that measure vital signs (analogue) and send digital alerts to nurses' stations.

Advantages of Using Computers

  1. Process data faster than any other device
  2. More accurate and reliable results
  3. More efficient — require less effort to process data
  4. Quickly and effectively store and retrieve large amounts of data
  5. Very economical for storing information (conserve physical space)
  6. Occupy very little office space
  7. Help reduce paper work significantly
  8. Flexible — can perform a variety of jobs
  9. Can reduce the number of employees needed, lowering costs
  10. Enhance security and confidentiality through password protection
  11. Have made communication easier
  12. Produce tidy, accurate output
  13. Reduce problems of data duplication
  14. Can operate in risky environments dangerous to humans

Disadvantages of Using Computers

  1. Very costly to purchase and maintain
  2. Can only be used where there is a source of power
  3. Require skilled manpower to operate
  4. Records are kept in a form not visible or human-readable
  5. Can break down like any other machine
  6. Information stored can be lost due to power interruptions or breakdown
  7. Do not have their own intelligence — can only work as instructed
  8. May cause retraining or retrenchment of staff
  9. Technology changes rapidly, causing computers to become obsolete quickly
  10. Have increased unemployment — computers now do jobs previously done by humans
  11. Increased computer crimes, particularly in banking

  • Supermarkets — stock control, receipts, barcode readers, change calculation.
  • Industries — process monitoring, robots, management control, advertising.
  • Banks/Insurance — ATMs, cheque processing, payrolls, electronic money transfer.
  • Process Control — controlling chemical and mechanical processes in factories.
  • Hospitals — patient records, disease diagnosis, ICU life-support, medical equipment.
  • Offices — e-mails, fax, document production, record keeping.
  • Government — storing records, bills, statements, improving civil service efficiency.
  • Education — Computer Aided Learning (CAL), simulations, open university, pilot training.
  • Research — scientific data analysis, engineering design, space exploration.
  • Communication — telephone exchanges, electronic messaging, fax.
  • Transport — traffic control, railways, shipping, airline reservations and radar.
  • Police / Law Enforcement — fingerprint databases, face recognition, criminal investigations.
  • Defence — radar systems, guided missiles, fighter jets, communication.
  • Multimedia — presentations, music synthesizers, entertainment, games, movies.
  • Domestic — watching movies, personal information, home budgets, shopping.
  • Library Services — record keeping, book catalogue searches.
  • Employment — computers have created many new employment opportunities.

Computer Laboratory

Definition

A computer laboratory is a room specially prepared to facilitate the installation of computers and provide a safe, conducive environment for teaching and learning Computer Studies.

Factors to Consider When Preparing a Lab

  • Security of computers, programs, and other resources
  • Reliability of the power source
  • Number of computers to be installed and floor space available
  • Maximum number of users the laboratory can accommodate

Requirements of a Computer Laboratory

  • Standard and sufficient furniture
  • Good ventilation
  • Reliable and sufficient power supply
  • Free from dust and moisture
  • Sufficient floor space
  • Proper cabling of electric wires
  • Fire-fighting equipment
  • Good lighting equipment
  • Strong rooms and doors for security

Safety Precautions in a Computer Laboratory

1. Behaviour in the Lab

  • Only authorized people should enter the computer room
  • Remove shoes before entering to prevent dust
  • No smoking; avoid exposing computers to dust
  • Do not carry food, drinks, or beverages into the computer room
  • Avoid unnecessary movements
  • Computer illiterates should not operate computers unsupervised
  • Always follow proper procedures for starting and shutting down computers

2. Protection Against Fire and Accidents

  • Keep inflammable chemicals stored away after use
  • Ensure all electrical wires are properly insulated
  • Always use a gaseous (CO₂) fire extinguisher — never water or powder
  • Report any incidents to the person in charge
Important
Water-based or powder extinguishers should NOT be used in a computer room. Water causes rusting and short circuits; powder particles scratch storage devices during read/write operations.

3. Insulation of Cables

All power cables must be properly insulated and laid away from busy pathways (preferably along walls) to prevent stumbling, electric shock, or power interruptions.

4. Stable Power Supply

Computers require a stable source of power. Use a UPS (Uninterrupted Power Supply) or Stabilizer to protect against surges and brownouts.

Functions of the UPS

  1. Regulates power from an unstable source to a clean, stable voltage
  2. Prevents power surges and brownouts
  3. Temporarily provides power during main power failure (allows the user to save work and shut down correctly)
  4. Alerts the user of any power loss (by beeping)

5. Burglar Proofing

  • Fit strong metallic grills and locks on doors, windows, and roofs
  • Use personal identification cards and fingerprint identification
  • Install security alarms at strategic access points
  • Protect computers with passwords

6. Ventilation

The computer room must have good air circulation to avoid overheating. Ensure large windows and doors, install air-conditioning systems and cooling fans, and avoid overcrowding.

7. Dust Control

  • Set up the lab away from excessive dust
  • Remove shoes before entering
  • Fit special curtains to reduce dust entry
  • Cover floors with carpets to absorb dust and noise
  • Use dust covers when computers are not in use

8. Damp Control

Maintain humidity at an optimum 50%. Low humidity allows static electricity build-up; high humidity (over 70%) causes rusting. Use humidifiers or dehumidifiers as needed.

9. Handling Materials & Equipment

  • Handle devices with great care — dropping or bumping causes permanent damage
  • Use the manufacturer's shipping carton when transporting devices
  • Do not place heavy objects on computers
  • Protect devices from direct sunlight and electrostatic discharge
  • Regularly check and service equipment

10. Standard Furniture & Posture

  • Table must be strong and wide enough to hold all equipment
  • Seat must be comfortable with a straight backrest
  • Forearms should be parallel to the floor; wrists straight
  • Eyes at the same level as the top of the screen
  • Feet firmly flat on the floor

11. Lighting & Vision Care

  • The room must be well lit to avoid eyestrain
  • Tilt the display away from windows to minimize glare
  • Use radiation filter screens and indirect lighting
  • Avoid using a flickering monitor
  • Use brightness and contrast controls to adjust the display
  • Take frequent breaks and rest your eyes
  • Distance between user and screen: 450–500 mm

Starting Up (Booting) a Computer

  1. Ensure all components are properly connected and the computer is connected to an active power source.
  2. Turn on the switch at the power source; if using a UPS or stabilizer, turn it on after the main supply.
  3. Switch on the Monitor first, then the System Unit. The computer will automatically begin booting.
Booting

Booting is the process of starting up a computer — the entire process that makes the computer ready for use.

Types of Booting

Cold Booting

Happens when a computer that was originally OFF is switched on by pressing the power button on the system unit.

Warm Booting

Happens when a computer that was already ON is restarted by pressing the Restart button or using the keyboard shortcut Ctrl + Alt + Del.

POST — Power-On Self-Test

When power is switched on, the computer executes the POST program stored in ROM. POST checks RAM, CMOS (BIOS), hard disk controller, floppy disk drive controller, and keyboard. If a device is faulty or missing, an error message or abnormal beep codes are produced.

The program that directs POST is called the BIOS (Basic Input Output System).

Shutting Down a Computer

  1. Save all work and close all running programs.
  2. Remove any floppy disk from the drive.
  3. Follow the proper shut-down procedure (e.g., Start → Shut Down in Windows).
  4. After the system powers down, switch off the Monitor, Printer, and any other output devices.

The Keyboard

The keyboard is a computer input device through which data and instructions are typed into the computer memory.

Types of Keyboard

  • Standard Keyboard — has 99 keys
  • Enhanced Keyboard — has between 102 and 105 keys

Sections of the Keyboard (101 keys total)

a. Function / Command Keys (F1–F12)

Located along the top of the keyboard. Used to issue commands; their function varies with different applications. Also called Programmable Keys.

b. Alphanumeric Keys

Contains 26 letters (A–Z), number keys (0–9), punctuation marks, and symbols. The Space Bar at the bottom is used to separate words.

c. Numeric Keypad Keys

Located on the rightmost part. Contains digits 0–9 and mathematical symbols (*, -, +, /, .). Used for fast numeric data entry. Active only when Num Lock is on.

d. Directional / Cursor Positioning Keys

  • Arrow keys — move the cursor one character/line in any direction
  • Page Up / Page Down — move the cursor one page up or down
  • Home / End — move the cursor to the beginning or end of the current line

Editing Keys

  • Backspace (←) — erases characters to the left of the cursor
  • Delete (Del) — erases characters to the right of the cursor
  • Insert (Ins) — switches between Insert mode and Overtype mode

e. Special PC Operation Keys

  • Tab (⭾) — moves the cursor at set intervals on the same line
  • Caps Lock — toggles between uppercase and lowercase letters
  • Shift (⇧) — used for single capital letters or symbols above number keys
  • Enter (⏎) — moves to a new line; executes selected commands
  • Escape (Esc) — cancels or quits a task
  • Control (Ctrl) — used in combination with other keys for special functions (e.g., Ctrl+S to save)

Practical Keyboard Skills

  1. Sit upright with both feet firmly on the ground.
  2. Place material to be typed on your left in a readable position.
  3. Rest both hands on the keyboard with fingers on the Home Keys: Left hand — A, S, D, F; Right hand — J, K, L, ; (semicolon). Thumbs on the Spacebar.
  4. Start typing slowly, using all ten fingers, pressing the key nearest to the home keys with the closest finger.

The Mouse

A pointing device that enables the user to issue instructions to the computer by controlling a special mouse pointer on the screen.

Parts of a Mouse

  • Casing — for holding the mouse in the hand
  • Roller ball — slides on a flat surface; moves the cursor on screen
  • Sensor Buttons (Left & Right) — used for making selections
  • Cable — connects the mouse to the system unit

Mouse Terminology

  • Point — move the mouse until the pointer tip is over the desired item.
  • Click — press and release the left button once; typically selects an item.
  • Double-click — press the left button twice in quick succession; usually opens a file or starts a program.
  • Right-click — press the right button once; displays a Shortcut (Context-sensitive) Menu.
  • Drag and Drop — point to an item, hold left button, slide to new position, release button.

Rules for Using the Mouse

  • Place the mouse on a flat, smooth surface
  • Gently hold it with your right hand using the thumb and two rightmost fingers
  • Index finger rests on the left button; middle finger on the right button
