01Definition of a Spreadsheet

Definition

A spreadsheet is essentially a ledger sheet that lets you enter, edit and manipulate numeric data.

There are two types of spreadsheets:

Manual Spreadsheet
  • Used by bookkeepers as a ledger book
  • Sheets of paper divided into rows and columns
  • Amounts entered manually with pen or pencil
  • Limited in size and accuracy
Electronic Spreadsheet
  • Prepared using a computer program
  • Enter values in rows and columns like a manual sheet
  • Manipulate data mathematically using formulae
  • Far more powerful, accurate and flexible
Note

In these notes, the word spreadsheet refers to the electronic spreadsheet.

02Advantages of Electronic over Manual Spreadsheets

  1. Speed, accuracy and efficiency — utilises the powerful aspects of the computer to quickly accomplish tasks.
  2. Larger work area — a manual ledger has at most 30 columns and 51 rows; an electronic spreadsheet has at least 255 columns and 255 rows.
  3. Large storage space — utilises computer storage devices to save and retrieve documents.
  4. Neat work — all work is edited on screen; a final clean copy is printed. Neatness is not dependent on handwriting.
  5. Better formatting capabilities — far more options for presentation and layout.
  6. Built-in functions — inbuilt formulae (functions) enable quick mathematical data manipulation.
  7. Automatic recalculation — results of a formula are automatically updated when any referenced value changes. A manual sheet requires erasing and rewriting.

04Components of a Spreadsheet

1. Worksheet

The component in which data values are entered. Made up of rows (horizontal) and columns (vertical). The intersection of a row and a column is a cell. Each column is labelled with a letter and each row with a number. A cell is referenced by its column letter followed by its row number (e.g., B3). A group of worksheets makes up a workbook.

2. Database

Data values entered in worksheet cells can be managed using special Excel features (filtering, forms, subtotals, pivot tables, data validation). If related values for the same entity are entered on the same row, they form a record. A worksheet can therefore be used as a simple database.

// Example worksheet with records
Name    IDNo.   Age   Nationality
Joy     123445   25    Kenyan
Jeff    125675   35    Ugandan

3. Graphs / Charts

A pictorial representation of data on a worksheet. Charts make it easy to present complex data in a simple, easy-to-understand format. Examples include pie charts, line charts, and bar charts.

05Application Areas of a Spreadsheet

Area 1

Statistical Analysis

  • Data analysis tools for statistical and engineering analyses
  • Functions: Average, Median, Mode, etc.
  • Some tools generate charts alongside output tables
Area 2

Accounting

  • Recording daily transactions and financial records
  • Tracking asset depreciation/appreciation
  • Calculating profits and preparing budgets
Area 3

Data Management

  • Neat tabular arrangement of data
  • Sorting, filtering, using forms
  • Linking worksheets for accessibility
Area 4

Forecasting ("What if")

  • Change one value to see its effect on results
  • Financial forecasting, budgeting, stock portfolio analysis
  • E.g., "What if sales increase by 20%?"

09Creating a Worksheet

Using the General Format

When the spreadsheet program starts, a new blank screen of rows and columns is presented. Enter data directly into the cells. To create a new blank workbook: File Tab → New → Blank Workbook.

Using a Template

Click File Tab → New, then select from available templates (Agendas, Budgets, Calendars, Expense Reports, Forms, Invoices, etc.) and double-click to open.

NB

If the template was saved previously on the hard disk, it opens as a new worksheet with preformatted features ready for data entry. Some templates may require the original installation disk.

10Editing a Cell Entry

Editing a cell means changing its contents. The cell must first be selected (made the current cell).

  1. Move the cell pointer to the cell you wish to edit.
  2. Double-click the formula bar for the text cursor to appear; the Status bar changes to "Edit".
  3. Use the keyboard to delete and add content in the formula bar, then press Enter to apply.
  4. Click the Save button on the Quick Access Toolbar or press Ctrl+S to save.

11Selecting a Range

A range is a rectangular arrangement of cells specified by the top-left and bottom-right cell addresses, separated by a colon (e.g., A1:C10).

Selecting Multiple Ranges

Shift Key
  • Highlights all columns/rows between the first selected cell and the newly clicked cell
Ctrl Key
  • Selects individually clicked cells or ranges without deselecting others

Hiding Rows / Columns

  1. Highlight the columns/rows you want to hide.
  2. Right-click on the selected column/row header.
  3. Click the Hide command from the context menu.

12Saving, Closing & Exiting

Saving a Worksheet

  1. Click File Tab → Save As (or click the Save icon on the Quick Access Toolbar).
  2. Select the save location in the Save in box; type a unique name in the File name box. Ensure Microsoft Excel Workbook is selected as the file type.
  3. Click Save.

Retrieving a Saved Workbook

  1. Click File Tab → Open.
  2. Select the drive or folder where the workbook was saved.
  3. Double-click the workbook icon to open it.

Closing & Exiting

  • Close: File → Close — closes the worksheet but keeps Excel open.
  • Exit: File → Exit — closes both the worksheet and the Excel program.

13Cell Data Types

Type 1

Labels

  • Any text or alphanumeric characters
  • Used as row/column headings
  • Left-aligned by default
  • Cannot be manipulated mathematically
  • Prefix apostrophe (') to treat a number as a label (e.g., '1990)
Type 2

Values

  • Numbers that can be manipulated mathematically
  • Includes currency, dates, numbers (0–9) and special symbols
  • Right-aligned by default
Type 3

Formulae

  • User-designed mathematical expressions
  • Must begin with = in Excel
  • Create relationships between cells
  • E.g., =B3+D4
  • Auto-recalculates when referenced cells change
Type 4

Functions

  • Inbuilt predefined formulae
  • Cover common calculations
  • E.g., =SUM(B3:D4)
  • Faster than creating custom formulae

14Cell Referencing

A cell reference identifies a cell or range on the worksheet, telling Excel where to look for values used in a formula.

A1 vs R1C1 Referencing Styles

A1 StyleR1C1 StyleMeaning
B2R2C2Row 2, Column 2
C10R10C3Row 10, Column 3
E20R20C5Row 20, Column 5

The R1C1 style is useful when automating tasks using Macros.

Relative Referencing

Cell references that change automatically depending on their position when a formula is copied. For example, typing =A1+B1 in cell C1 and copying it to C2 automatically changes it to =A2+B2.

Absolute Referencing

Cell references that always refer to a specific fixed location, even when copied. Add a dollar sign ($) before the column letter and/or row number to make it absolute.

// Absolute reference — does not change when copied
=$B$10   // Both column B and row 10 are fixed
=$B10   // Only column B is fixed; row changes
=B$10   // Only row 10 is fixed; column changes

Referencing Using Labels and Names

You can create a named range to use descriptive names instead of cell addresses in formulae.

// Using a named range
=SUM(SecondQuarterProfits)
// Easier to understand than:
=SUM(A10:C20)

Creating a Named Range

  1. Select the range to be named.
  2. Click inside the Name Box (top left); delete the cell reference and type a descriptive name.
  3. Press Enter to apply.

16Arithmetic Operators & Order of Execution

Operators
  • + — Addition
  • - — Subtraction
  • * — Multiplication
  • / — Division
  • ( ) — Parentheses (calculated first)
Operator Precedence
  1. Negation (−)
  2. Percent (%)
  3. Exponentiation (^)
  4. Multiplication and Division (* /)
  5. Addition and Subtraction (+ −)
  6. Relational (=, <>, >, <, <=, >=)
Tip

Enclose part of a formula in parentheses to force it to be calculated first. For example: =(A1+C3)/E20 adds A1 and C3 before dividing by E20.

17Copying and Moving Data

When data is cut or copied, it is temporarily held in a storage location called the clipboard.

Copying Data
  • Creates a duplicate — original remains unchanged
  1. Highlight the cells/range to copy
  2. Click Copy on the Home tab
  3. Select the destination cell
  4. Click Paste on the Home tab
Moving Data
  • Transfers contents to a new location — original is removed
  1. Highlight the range to move
  2. Click Cut on the Home tab
  3. Select the destination cell
  4. Click Paste on the Home tab

18Worksheet Formatting

Formatting enhances the appearance of a worksheet to make it more attractive and readable. The golden rule: use simple, clear formats.

Formatting Text

  1. Highlight the cells containing the text to format.
  2. Click the dialog box launcher under the Font group on the Home tab.
  3. Select the Font tab and choose font type (e.g., Times New Roman), style, size, underline, and colour.
  4. Click OK to apply.

Formatting Numbers

  1. Highlight the cells with the numbers to format.
  2. Open the Format Cells dialog → select the Number tab.
  3. Choose from the available formats:
FormatUse
GeneralNo specific number format
NumberGeneral display of numbers (e.g., 2345.23)
CurrencyMonetary values (e.g., $100, Ksh.10)
AccountingLines up currency symbols and decimal points
DateDisplays date in chosen format
TimeDisplays time in chosen format
PercentageMultiplies cell value by 100 and displays with %
TextTreats cell content as text, even if it's a number
CustomDefine a number format not predefined in Excel

Worksheet Borders

  1. Highlight the range to add borders.
  2. Open Format Cells dialog → click the Borders tab.
  3. Specify border options (left, right, top, bottom); select line thickness and style.
  4. Click OK.

19Formatting Rows and Columns

The standard column width in Excel is 8.43 characters (adjustable between 0–255).

Changing Column Width

  • Drag method: Move the mouse to the right-hand border line between column headers; the pointer becomes a double arrow — drag to the desired width.
  • Menu method: Click the column header → Right-click → Column Width → type a value → OK.

Changing Row Height

  • Drag method: Point to the line separating two row numbers; the pointer becomes a double arrow — drag to the required height.
  • Menu method: Click the row header → Right-click → Row Height → type a value → OK.

Inserting Rows

  1. Click a cell (e.g., A5) to make it active.
  2. Home tab → Insert dropdown → Insert Sheet Rows. A row is inserted above A5; all other rows shift down.

Inserting Columns

  1. Click a cell (e.g., A5) to make it active.
  2. Home tab → Insert dropdown → Insert Sheet Columns. A column is inserted to the left; all others shift right.

20Global Worksheet Formatting

To format the entire worksheet at once, first select it globally using one of these methods:

  • Click the blank corner cell (top-left, just left of column A and above row 1)
  • Press Ctrl+A

The whole worksheet becomes highlighted. Apply Format Cells settings to format it as one block.

21Sorting

Arranges records in a particular order (ascending or descending) for easier data access.

  1. Highlight the range to sort by clicking its column header letter.
  2. Click the Data tab → Sort.
  3. In the Sort dialog: select the field (criteria), the sort key, and sort order (Ascending/Descending).
  4. Click OK to apply.

22Filtering Data

Filtering displays only the rows that meet a specified condition. Excel has two filter commands:

AutoFilter
  • Uses simple criteria including filter by selection
  • One filter per worksheet at a time
Advanced Filter
  • Uses more complex, multi-condition criteria

Using AutoFilter

  1. Click a cell in the list/column to be filtered.
  2. On the Data tab, click Filter.
  3. Click the dropdown arrow in the column containing the data.
  4. Select the value to display from the dropdown list.

23Subtotals Function

Automatically summarises data by calculating subtotal and grand total values for a sorted list.

Requirement

The list must have labelled columns and must be sorted on the column for which you want subtotals.

  1. Click a cell in the list (e.g., A3).
  2. On the Data tab, click Subtotals. The entire data range is selected.
  3. In the At each change in box, select the grouping field (e.g., Name).
  4. In the Use function box, select the function (e.g., Sum).
  5. In the Add subtotals to box, check the appropriate field (e.g., Amount Owed).
  6. Click OK — subtotals are inserted automatically.

AutoCalculate / Totals

To quickly find the total of a range, highlight it then click the AutoSum (∑) icon on the Standard toolbar. Right-clicking the sum on the Status Bar also shows Min, Max and Average.

24Forms

A form is a specially prepared template allowing users to enter data in a more convenient format. If data is first collected on paper, a form can mirror that paper layout to speed up data entry.

To display a form: Data → Form.

25Charts / Graphs

Charts are graphics that represent values and their relationships, helping readers quickly see trends and comparisons. Excel supports both 2D and 3D charts.

Types of Charts

Chart TypeDescription / Best Use
Line ChartRepresents data as lines with markers at each data value. Best for showing trends over time (e.g., student performance Form 1–3).
Column ChartClusters of vertical columns comparing values across categories.
Bar ChartData values arranged horizontally; compares values across categories.
Pie ChartDisplays the contribution of each value to a grand total (percentage view).
Scatter ChartCompares pairs of values on the same axis.
Bubble ChartLike a scatter chart but with a third data dimension shown by bubble size.
Area ChartLike a line chart but with the area below the line filled in.

Creating a Chart

  1. Select the range of values for the chart.
  2. Click the Insert tab → in the Charts group, click the chart type of your choice (or click the dialog box launcher to see all chart types).
  3. Select the All Charts tab and choose the chart sub-type.
  4. Click OK — the chart appears on the spreadsheet.

Moving and Resizing a Chart

  • Resize: Click the chart object and drag the handles at its edges/corners.
  • Move: Click inside the chart area and drag it to the desired position.

Data Ranges

A data range is the rectangular block of cells providing the base data for the chart. Referenced as an absolute range (e.g., =Sheet1!$B$2:$C$8). To view: right-click the chart → Source Data.

Labels, Titles and Legends

To add any chart element (data labels, chart title, axis titles, legend):

  1. Select the chart.
  2. Click the + (plus) icon on the right side of the chart to display chart elements.
  3. Check the desired element (Data Labels, Chart Title, Axis Titles, Legend) and edit as needed.
Legend

Like a map key — explains what each colour or pattern in the chart represents. Essential when a chart has multiple data series.

26Printing Worksheets

Page Setup

  1. Click Page Layout tab → Page Setup dialog box launcher.
  2. On the Page tab, select orientation (Portrait or Landscape) and other settings.
  3. Click OK to apply.

Page Orientation

Portrait
  • Taller than wide (standard)
  • Best for worksheets with few columns
Landscape
  • Wider than tall
  • Best for worksheets with many columns

Print Options

Click File Tab → Print to open the Print dialog.

  • Select Printer — choose from all installed printers.
  • Selection — prints only the selected worksheet area.
  • Workbook — prints all worksheets in the workbook.
  • Selected Chart — prints only the selected chart.
  • Number of Copies — specify how many copies to print.
  • Page Range — specify pages to print (e.g., pages 50 to 60 from a 100-page workbook).

Common Printing Problems & Solutions

ProblemPossible Cause & Solution
