Introduction

ICT Career Overview

ICT has created an entirely new landscape of job titles and professional roles. Below are the major career opportunities available in the field of Information and Communication Technology.

System Analyst Software Engineer Computer Engineer Web Designer Web Administrator Computer Operator Computer Technician Data Processing Manager IS Manager Computer Trainer Network Administrator Graphics Designer Typesetter Librarian Chief Programmer Database Administrator Computer Programmer
Career Role 01

System Analyst

A person responsible for analysing a company's needs or problems, then designing and developing a computer-based information system.
Qualities
  • Good problem-solving skills and creativity
  • Good communication skills
  • Business knowledge
  • Trained as an analyst
  • General knowledge of the firm's goals and objectives
  • Knowledge of data processing methods and current hardware
  • Familiarity with available programming languages
Duties
  • Gather, record and analyse facts of systems
  • Design new systems and recommend changes
  • Prepare instruction manuals
  • Co-ordinate training for users of new systems
  • Work with programmers to construct and test systems
  • Prepare system specifications
  • Co-ordinate implementation of new or modified systems
Career Role 02

Chief Programmer / Programming Manager

Reports to the Data Processing Manager on activities of the programming group, including junior and senior programmers.
Duties
  • Managing the programming team
  • Liaising with the Chief System Analyst
  • Reviewing and controlling program documentation
  • Reviewing program performance
  • Ensuring all programs are thoroughly tested before deployment
  • Reviewing all system specifications before handover to programmers
Career Role 03

Computer Programmer

People whose work entails coding — formulating instructions for the computer to solve given problems.
Qualities
  • Good command of the programming language in use
  • Knowledge of programming methodology and hardware relationships
  • Creativity for developing new problem-solving methods
  • Patience and persistence
Duties
  • Coding programs using appropriate programming languages
  • Testing for logical, syntax and semantic errors
  • Debugging programs
  • Maintaining and reviewing existing programs
  • Developing customised application and system programs
System Analyst vs Programmer: A system analyst is primarily responsible for analysing weaknesses in the existing system and designing an alternative, while a programmer mainly writes computer programs based on system requirements.
Career Role 04

Database Administrator (DBA)

A person responsible for the design and implementation of the organisation's database.
Duties
  • Responsible for the design and control of the organisation's database
  • Enforcing security measures through controls and monitoring database usage via user passwords
  • Keeping the database up to date — adding new records, modifying or deleting unnecessary ones
  • Establishing the appropriate content and format of data records
Career Role 05

Software Engineer

A person skilled in software development and the technical operation of computer hardware.
Duties
  • Developing system and application software
  • Developing user and technical documentation for new software
  • Maintaining and updating software to meet day-to-day requirements
Career Role 06

Computer Engineer

Specialised personnel responsible for assembling computer systems and developing new technologies in ICT. Must have a strong background in electronic engineering.
Duties
  • Design and develop computer components (e.g. motherboards, storage devices)
  • Design and develop computer-controlled engineering devices such as robots
  • Re-engineer computer components to enhance functionality and efficiency
  • Determine the electrical power requirements of each computer component
Software Engineer vs Computer Engineer: A software engineer develops software and programs as per requirements. A computer engineer designs computer hardware and improves on existing components.
Career Role 07

Web Designer & Web Administrator

Web Designer

Designs websites using various software tools. Websites contain pages where individuals and organisations can advertise themselves and promote products.
  • Developing, testing and uploading websites
  • Maintaining, updating and modifying website information
  • Monitoring access and use of internet connections
  • Downloading information needed by an organisation

Web Administrator

Manages websites by ensuring they are updated periodically to provide accurate information.
  • Developing, testing and uploading websites
  • Monitoring access and use of internet connections
  • Downloading information from the internet
  • Maintaining, updating and modifying website information
Career Role 08

Computer Operator

A person who works on a computer on a daily basis to perform data entry, fix minor problems, and provide general support within an organisation.
Duties
  • Entering data into the computer for processing
  • Keeping up-to-date records of all information processing activities
  • Responsible for the general equipment layout within the computer room
  • Activating the computer to obey programs through use of commands
  • Mounting storage media (e.g. disks) to their drives
  • Fixing computer-related problems such as booting issues, printer jams, and internet connection
Career Role 09

Computer Technician

A member of the technical staff whose job is to service and repair computers.
Duties
  • Troubleshooting computer hardware and software problems
  • Ensuring computer accessories (printers, storage media, etc.) are in good condition
  • Assembling and upgrading computers and components
  • Working with hardware engineers in designing computer components
Hardware Job Opportunities: Computer Engineer · Computer Repair & Maintenance · Computer Assembling
Career Role 10

Data Processing Manager

The person in charge of the data processing department within the organisation, ensuring that the organisation's needs are met.
Duties & Responsibilities
  • In charge of data and information flow within the department
  • Responsible for data collection and preparation
  • Purchase of hardware and other facilities, and their management
  • Software development and maintenance
  • Assessment and evaluation of staff performance
Career Role 11

Information System Manager

Controls, plans, staffs, schedules and monitors all activities of the ICT department in the organisation.
Responsibilities
  • Ensuring all tasks in the IT department are completed correctly and on time
  • Preparing budgets for department operations
  • Keeping the department inventory records up to date
  • Managing the human resource within the department
Career Role 12

Computer Teacher / Trainer

Specialised personnel in computing who are well conversant with various disciplines and computer-related issues, guiding those entering the field.
Duties
  • Training people on how to use computers and various application programs
  • Developing training reference materials
  • Guiding learners on how to acquire knowledge through research
  • Advising and preparing learners on best career opportunities in ICT
  • Preparing learners for ICT examinations
Career Role 13

Network Administrator

A specialist who oversees the smooth running of network systems within an organisation.
Responsibilities
  • Set up a computer network
  • Maintain and enforce security measures on the network
  • Monitor the use of network resources
  • Maintain and troubleshoot network-related problems
Career Role 14 & 15

Graphic Designer & Librarian

Computer Graphic Designer

Required to design graphical objects, illustrations, and professional publications such as books, newspapers, and magazines. Such professionals are employed in publishing organisations to typeset various printed media.

Librarian

Responsible for keeping files on various storage devices and ensuring the physical security of storage media.

  • Ensures enough physical files are available for use
  • Maintains external file labels
  • Ensures duplicate files are kept
  • Restricts file access to authorised persons only

Factors to Consider Before Joining an ICT College

  1. Accreditation — whether the institution offers ICT courses recognised locally and internationally. Accreditation is the process of recognising an institution as being able to offer degrees and diplomas.
  2. Cost of training — fees and associated expenses of studying at the institution.
  3. Location — proximity and accessibility of the institution.
  4. Physical management — the general condition, facilities, and management quality of the institution.
Section 03 — Revision

Review & Revision Questions with Answers

Explain the role of the following ICT specialists: Database Administrator, Software Engineer, System Analyst, Computer Technician, Information Systems Manager.
  • Database Administrator — Designs, develops, updates and maintains the information system database.
  • Software Engineer — Develops and updates software.
  • System Analyst — Analyses, designs and develops information systems.
  • Computer Technician — Troubleshoots computer hardware and software problems.
  • Information System Manager — Manages ICT operations within the organisation.
Outline the roles of a Network Administrator.
  • Setting up a computer network
  • Securing the network against unauthorised access
  • Managing the network to enhance performance and uptime
  • Monitoring use of network resources
List three types of job opportunities available in the field of computer hardware.
  • Computer Engineer
  • Computer Repair and Maintenance
  • Computer Assembling
Give examples of public and accredited private universities that offer computer-related courses in Kenya.
  • Public: Moi University, University of Nairobi, Kenyatta University, JKUAT, Maseno University, Egerton University
  • Private: Catholic University, Daystar University, Baraton University, Kabarak University
Identify ICT-related courses offered at University, Polytechnic, and College level.
  • University: BSc. Computer Science · BSc. Information Technology · MSc. Information Systems
  • Polytechnic: Diploma in Information Technology · Diploma in Computer Hardware Maintenance · Diploma in Computer Programming
  • College: Diploma in Information Technology · Certificate in Information Technology · Diploma in Education (Computer Science)
Define the term "accreditation" as used in education.
Accreditation is the process of recognising an institution as being able to offer degrees and diplomas.
State factors to consider before enrolling for an ICT course in a college.
  • Cost of the course
  • Job opportunities associated with the qualification
  • Whether the examinations offered are recognised locally and internationally
  • The organisation/body offering the course
